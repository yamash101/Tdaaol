ربط حساب باينانس

للبدء في التداول، يرجى ربط حساب باينانس الخاص بك.
هام جداً: لا تفعل صلاحية السحب (Withdrawal) عند إنشاء مفتاح الـ API!

خطوات الحصول على مفتاح الـ API من باينانس:

  1. قم بتسجيل الدخول إلى حسابك في باينانس.
  2. اذهب إلى "الملف الشخصي" ثم "إدارة API" (API Management).
  3. انقر على "إنشاء مفتاح API" (Create API Key).
  4. قم بتسمية المفتاح (مثلاً: "منصتي الخاصة").
  5. تأكد من تفعيل صلاحيتي "القراءة" (Enable Reading) و "التداول الفوري" (Enable Spot & Margin Trading) فقط.
  6. لا تفعل صلاحية "السحب" (Enable Withdrawals) أبداً للحفاظ على أمان أموالك.
  7. انسخ مفتاح الـ API (API Key) ومفتاح السري (Secret Key) الظاهرين لك.
