ربط حساب باينانس خطوتك الأولى للتداول

للبدء في رحلة التداول الممتعة والآمنة، يرجى ربط حساب باينانس الخاص بك بمنصتنا.
تنبيه هام جداً للأمان: عند إنشاء مفتاح الـ API في باينانس، **لا تفعل صلاحية السحب (Enable Withdrawals) أبداً!** هذا يحمي أموالك.

خطوات سهلة للحصول على مفتاح الـ API من باينانس:

  1. قم بتسجيل الدخول إلى حسابك في باينانس.
  2. اذهب إلى "الملف الشخصي" (Profile) ثم "إدارة API" (API Management).
  3. انقر على زر "إنشاء مفتاح API" (Create API Key).
  4. قم بتسمية المفتاح (مثلاً: "منصتي الخاصة").
  5. تأكد من تفعيل صلاحيتي "القراءة" (Enable Reading) و "التداول الفوري" (Enable Spot & Margin Trading) فقط.
  6. تكرار هام جداً: لا تفعل صلاحية "السحب" (Enable Withdrawals) أبداً للحفاظ على أمان أموالك.
  7. انسخ مفتاح الـ API (API Key) ومفتاح السري (Secret Key) الظاهرين لك.
